#-------------------------------------------------------
# Stability script
#
# Version: 1.8
# Author: Dimitrios Simatos
# Last updated: 2019-05-24
#-------------------------------------------------------
# Description
#
# This script is designed to pump a liquid from a single port into the flow cell.
#-------------------------------------------------------
# Changes
#
# Added final cycle of flushing the device with fresh liquid.
#-------------------------------------------------------
# Important points
#
# Never leave any liquid in the pump to dry after the experiments. If they dry, the residue may block the valve.
# Always run the cleaning cycle after the experiments.
# Use clockwise rotation when going up in valve port numbers and vice versa.
# Do not tighten the reservoir caps. It may lead to overpressure of air.
#-------------------------------------------------------
# Things to do before running the script
#
# Set time units to minutes.
# Set filenames.
# Replace the tubing, if necessary.
#-------------------------------------------------------
# Port connections
#
# Port 1: Waste
# Port 2: Device
# Port 3: Ultrapure water
# Port 4: Nothing
# Port 5: Nothing
# Port 6: Nothing
#-------------------------------------------------------
# Pumping cycles and execution time
#
# There is only 1 step, with 90 cycles.
# There are 90 cycles, each one followed by 8 minutes of measurements.
# Execution time: 1 x 90 x 8 = 720 minutes = 12 hours
# Only the first cycle pumps ultrapure water from port 3.
# The first cycle has the first report valve command commented.
# The last cycle is similar to the first cycle (with the filename command removed and the valve commands uncommented). It flushes the device with fresh liquid before the end of the measurement.
#-------------------------------------------------------
# Volume of liquid needed
#
# Volume of liquid needed per cycle: 1 ml
# Total volume of liquid needed per script run:
# 1 ml ultrapure water
#-------------------------------------------------------
# Parameters
#
# Syringe size: 1 ml
# Step resolution N = 0.01 mm
# Drawing flow V = 100 pulses/second (2000 ul/min)
# Pumping flow: Initially 10 pulses/second (200 ul/min), then static.
#-------------------------------------------------------
# SPA settings
itime l 0.04
gaterangemode l
drainrangemode l
#-------------------------------------------------------
# Pump initialization
pump /1T #hard stop pump
pump /1ZR #initialize pump
#-------------------------------------------------------
# Cycle 1: Pumping the liquid of port 3 (8 minutes)
filename step_1_up-H2O_initial_
pump /1Q #query pump
pump /1? #report absolute plunger position
#pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute
pump /1N0R #set step resolution: 0.01 mm
pump /1Q #query pump
pump /1? #report absolute plunger position (the plunger should be at 0)
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait 1000 #wait 1 second
pump /1V100R #choose speed: 100 steps/s (200 ul/min)
pump /1Q #query pump
wait 1000 #wait 1 second
pump /1I3R #go to port 3, clockwise rotation
pump /1Q #query pump
pump /1?6 #report valve position
wait 1000 #wait 1 second
pump /1A3000R #go to absolute position 3000 (= draw 1 ml)
wait-min 1 #wait 1 minute
pump /1V10R #set speed to 10 steps/s (200 ul/min)
pump /1Q #query pump
pump /1? #report absolute plunger position (the plunger should have finished moving)
#pump /1?6 #report valve position
pump /1O2R #switch to port 2, counterclockwise rotation
pump /1Q #query pump
pump /1?6 #report valve position
pump /1A100R #go to absolute position 100 (do not fully empty syringe)
pump /1Q #query pump
pump /1? #report absolute plunger position (the plunger should be moving to 0)
#pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute
pump /1Q #query pump
pump /1? #report absolute plunger position (the plunger should be moving to 0)
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute
pump /1Q #query pump
pump /1? #report absolute plunger position
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute
pump /1Q #query pump
pump /1? #report absolute plunger position
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute
