#-------------------------------------------------------

# Version: 1.5
# Author: Dimitrios Simatos
# Last updated: 2019-01-21
#-------------------------------------------------------
# Description
#
# This script is designed to perform a cycling experiment between four liquids.
#-------------------------------------------------------
# Changes
#
# Removed 1A0R step. Syringe should not fully empty as air is injected.
#-------------------------------------------------------
# Important points
#
# Never leave any liquid in the pump to dry after the experiments. If they dry, the residue may block the valve.
# Always run the cleaning cycle after the experiments.
# Use clockwise rotation when going up in valve port numbers and vice versa.
# Do not tighten the reservoir caps. It may lead to overpressure of air.
#-------------------------------------------------------
# Things to do before running the script
#
# Set time units to minutes.
# Set filenames.
# Replace the tubing, if necessary.
#-------------------------------------------------------
# Port connections
#
# Port 1: Waste
# Port 2: Device
# Port 3: Ultrapure water
# Port 4: Phosphate buffer (pH=4.07 - 50 mM)
# Port 5: Phosphate buffer (pH=3.04 - 50 mM)
# Port 6: Phosphate buffer (pH=2.04 - 50 mM)
#-------------------------------------------------------
# Pumping cycles and execution time
#
# There are 7 steps, with 4 cycles per step.
# Execution time: 28 x 30 = 840 minutes = 14 hours
# There are 4 pumping cycles, each one followed by 30 minutes of measurements.
# Cycles 1-4, 9-12, 17-20, 25-28 pump ultrapure water from port 3.
# Cycles 5-8 pump phosphate buffer (pH=4.07 - 50 mM) from port 4.
# Cycles 13-16 pump phosphate buffer (pH=3.04 - 50 mM) from port 5.
# Cycles 21-24 pump phosphate buffer (pH=2.04 - 50 mM) from port 6.
# Within a step all four cycles are similar, but the first cycle has the initialization command uncommented and the first report valve command commented.
#-------------------------------------------------------
# Volume of liquid needed
#
# Volume of liquid needed per cycle: 1 ml
# Total volume of liquid needed per script run:
# 16 x 1 = 16 ml ultrapure water
# 4 x 1 = 4 ml phosphate buffer (pH=2.04 - 50 mM)
# 4 x 1 = 4 ml phosphate buffer (pH=3.04 - 50 mM)
# 4 x 1 = 4 ml phosphate buffer (pH=4.07 - 50 mM)
#-------------------------------------------------------
# Parameters
#
# Syringe size: 1 ml
# Step resolution N = 0.01 mm
# Drawing flow V = 100 pulses/second (2000 ul/min)
# Pumping flow V = 2 pulses/second (40 ul/min)
