#-------------------------------------------------------
# Stability script
#
# Version: 1.9
# Author: Dimitrios Simatos
# Last updated: 2022-09-08
#-------------------------------------------------------
# Description
#
# This script is designed to pump a liquid from a single port into the flow cell.
# The script allows the baseline 2 hours to stabilize and then flushes the flow cell with liquid.
# The script was made to see if the baseline stability depends on liquid being flushed inside the flow cell
# (and hence if the liquid's conductivity changes with time if it remains still inside the flow cell).
#
# Input port sequence: 3
#-------------------------------------------------------
# Changes
#
# Added input port sequence and updated the volume of liquid needed.
#-------------------------------------------------------
# Important points
#
# Never leave any liquid in the pump to dry after the experiments. If they dry, the residue may block the valve.
# Always run the cleaning script after the experiments.
# Use clockwise rotation when going up in valve port numbers and vice versa.
# Do not tighten the reservoir caps. It may lead to overpressure of air.
#-------------------------------------------------------
# Things to do before running the script
#
# Set time units to minutes.
# Set filenames.
# Replace the tubing, if necessary.
#-------------------------------------------------------
# Port connections
#
# Port 1: Waste
# Port 2: Device
# Port 3: Ultrapure water
# Port 4: Nothing
# Port 5: Nothing
# Port 6: Nothing
#-------------------------------------------------------
# Pumping cycles and execution time
#
# There are 6 steps in total.
# Every step consists of 16 cycles: one pumping cycle and two hours of measurements (15 cycles).
# Every 2 hours (15 cycles) new liquid is flushed inside the flow cell.
# The water takes 30 minutes to flow.
# Every cycle is 8 minutes long. The total number of cycles is 96.
# Execution time: 6 x 16 x 8 = 768 minutes = 12.8 hours
#-------------------------------------------------------
# Volume of liquid needed
#
# Volume of liquid needed per step: 1 ml
# Total volume of liquid needed per script run (volume/cycle * cycles in which the liquid is injected):
# 1 ml x 6 = 6 ml ultrapure water
#-------------------------------------------------------
# Parameters
#
# Syringe size: 1 ml
# Step resolution: N = 0.01 mm
# Drawing flow: V = 100 pulses/second (2000 ul/min)
# Pumping flow: V = 2 pulses/second (40 ul/min)
# NOTE: The first cycle has a pumping flow of 10 pulses/second (200 ul/min). This is done to increase the pressure inside the flow cell, and ensure that there are no air pockets during the initial filling.
#-------------------------------------------------------
# SPA settings
itime l 0.04
gaterangemode l
drainrangemode l
#-------------------------------------------------------
# Pump initialization
pump /1T #hard stop pump
pump /1ZR #initialize pump

# Cycle 1: Pumping the liquid of port 3 (8 minutes)

pump /1? #report absolute plunger position
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait-min 1 #wait 1 minute

pump /1Q #query pump
pump /1? #report absolute plunger position (the plunger should be at 0)
pump /1?6 #report valve position
transfer d 0 -1 -0.1 -1 -1 0
wait 1000 #wait 1 second
pump /1V100R #choose speed: 100 steps/s (2000 ul/min)
pump /1Q #query pump
wait 1000 #wait 1 second
pump /1I3R #go to port 3, clockwise rotation
pump /1Q #query pump
pump /1?6 #report valve position
wait 1000 #wait 1 second
pump /1A3000R #go to absolute position 3000 (= draw 1 ml)
wait-min 1 #wait 1 minute

pump /1? #report absolute plunger position (the plunger should have finished moving)
