#-------------------------------------------------------
# Liquid cycling script
#
# Version: 0.3
# Author: Dimitrios Simatos
# Last updated: 2022-11-16
# Changes: Updated the duration of script with the measured duration of script.
#-------------------------------------------------------
# Parameters
#
# Input port sequence: 3, 4, 3, 5, 3, 6, 3
# Outlet port sequence: 2
# Number of steps (step: an integer number of cycles during which a specific liquid is injected): 7
# Number of cycles: 112
# Cycles per step: 16
# Cycle time: 8 minutes
# Period (number of cycles between pumping cycles - even of the same liquid): 4
# Liquid injections per step (cycles per step/period): 4
# Duration of script: Theoretical: 15 hours, Measured: 
#
# Syringe size: 1 ml
# Step resolution: N = 0.01 mm
# Drawing flow (Qdraw): V = 100 pulses/second (2000 ul/min)
# Pumping flow (Qpump): V = 2 pulses/second (40 ul/min)
#  - NOTE: The first cycle has a pumping flow of 10 pulses/second (200 ul/min). This is done to increase the pressure inside the flow cell, and ensure that there are no air pockets during the initial filling.
#  - Time to empty syringe: Syringe size/Qpump = 25 minutes
#-------------------------------------------------------
# Description
#
# This script is designed to perform a cycling experiment between four liquids.
# If sensing proteins, perform two additional steps:
#  - Saturate the tubes for 1-2 hours with a highly concentrated protein solution.
#  - Measure consecutive transfer curves for 4 hours, to allow the device to charge and the baseline to stabilize to its maximum value.
#-------------------------------------------------------
# Important points
#
# Never leave any liquid in the pump to dry after the experiments. If they dry, the residue may block the valve.
# Always run the cleaning script after the experiments.
# Use clockwise rotation when going up in valve port numbers and vice versa.
# Do not tighten the reservoir caps. It may lead to overpressure of air.
#-------------------------------------------------------
# Things to do before running the script
#
# If sensing proteins, saturate the tubes with a highly concentrated protein solution.
# Set time units to minutes.
# Set filenames.
# Replace the tubing, if necessary.
#-------------------------------------------------------
# Port connections
#
# Port 1: Waste
# Port 2: Device
# Port 3: Ultrapure water
# Port 4: Lysozyme 0.1 g/L (100% Ultrapure water)
# Port 5: Lysozyme 1 g/L (100% Ultrapure water)
# Port 6: Lysozyme 10 g/L (100% Ultrapure water)
#-------------------------------------------------------
# Pumping cycles and execution time
#
# There are 7 steps in total.
# Every step consists of 16 cycles: one pumping cycle, 30 minutes of measurements (3 cycles), then another pumping cycle and another 30 minutes of measurements (3 cycles),
# then another pumping cycle and another 30 minutes of measurements (3 cycles), then another pumping cycle and another 30 minutes of measurements (3 cycles).
# Every 30 minutes (3 cycles) new liquid is flushed inside the flow cell.
# Every cycle is 8 minutes long. The total number of cycles is 112.
#
# Execution time for a single cycle: 8 minutes
# Total (theoretical) execution time: 1 (initialization) + 7 x 16 x 8 = 897 minutes = 15 hours
# Total (measured) execution time: 
#-------------------------------------------------------
# Volume of liquid needed
#
# Volume of liquid needed per step: 4 ml
# Total volume of liquid needed per script run: (volume/step * steps in which the liquid is injected)
# 4 ml x 4 = 16 ml ultrapure water
# 4 ml x 1 = 4 ml lysozyme 0.1 g/L (100% Ultrapure water)
# 4 ml x 1 = 4 ml lysozyme 1 g/L (100% Ultrapure water)
# 4 ml x 1 = 4 ml lysozyme 10 g/L (100% Ultrapure water)
